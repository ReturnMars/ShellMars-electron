# shellmars

基于 Electron + Vue + TypeScript 的跨平台桌面应用。

## 推荐开发环境

- [VSCode](https://code.visualstudio.com/) + [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## 项目脚本

### 安装依赖

```bash
pnpm install
```

### 本地开发

```bash
pnpm dev
```

### 代码检查

```bash
pnpm lint
```

### 项目构建

```bash
# Windows
pnpm build:win

# macOS
pnpm build:mac

# Linux
pnpm build:linux
```
